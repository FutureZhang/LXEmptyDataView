## 二 集成方式

1. Cocoapods方式集成: 
```
pod 'LXEmptyDataView'

使用时导入头文件：#import <LXEmptyDataView/LXEmptyDataView.h>
```

2. 手动下载集成: 
```
直接将LXEmptyDataView.h和LXEmptyDataView.m两个文件复制到你的工程中

使用时导入头文件：#import "LXEmptyDataView.h"
```
